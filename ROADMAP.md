# Project Roadmap

This roadmap outlines the future goals and milestones for the Live Dual Subtitle Translator project.

## Q3 2026
- [x] Initial release of Chrome Extension MVP
- [x] Integrate Gemini Translation API
- [ ] Implement Firefox Add-on support
- [ ] Add caching to reduce translation API costs

## Q4 2026
- [ ] Support for multiple translation providers (OpenAI, DeepL)
- [ ] Configurable hotkeys for toggling translations
- [ ] Improved UI/UX for the extension popup

## 2027 and Beyond
- [ ] Mobile browser support (Kiwi Browser, Firefox for Android)
- [ ] Custom styling options for subtitles (font, color, background)
- [ ] Offline translation capabilities (using local LLMs where possible)

> **Note:** This roadmap is subject to change based on community feedback and contribution. We welcome feature requests and PRs!
