# Contributing to Live Dual Subtitle Translator

First off, thank you for considering contributing to Live Dual Subtitle Translator! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/yashchowdhary14/live-dual-subtitle-translator/issues) first. If it's not there, feel free to open a new issue.

## Fork & create a branch

If this is something you think you can fix, then fork Live Dual Subtitle Translator and create a branch with a descriptive name.

## Get the test suite running

Make sure your changes don't break existing functionality. Run the following commands:

```bash
npm install
npm run build
npm run lint
```

## Implement your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

## Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with Live Dual Subtitle Translator's master branch. Then, push your branch to your fork and submit a Pull Request!

Please make sure you fill out our Pull Request template and provide enough context about your changes.
